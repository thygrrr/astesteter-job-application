# Astesteter

Playable builds:
https://tigerblue.itch.io/astesteter
(Win64 version recommended)

Due to the security CVE in Unity 2023 that this project was initially built on, I updated it to Unity 6.4 
(so as to not encourage installing a broken Unity editor)

Due to different storage limits on github, git lfs migrate was used to rewrite most of the repo's changesets.
